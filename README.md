## 9. 积木模式全键盘快捷键

- Alt+1：新增 朗读块

- Alt+3：新增 重复块
- Alt+4：新增 分节块
- Alt+5：新增 音效块

- Ctrl+↑ / Ctrl+↓：上移 / 下移选中块

- Delete / Backspace：删除选中块
- Space：预览播放
- Ctrl+S：保存项目
- F5：预览播放
- Esc：停止播放并关闭弹窗
